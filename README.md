# Screen Translator
A snipping tool like app, but for translating.


## Run
dotnet run

or

dotnet run -c Release

## Add AI model for translation
Create a folder called models and add

phi3-mini.gguf or mistral-v0.3.gguf

these are just the hardcoded model names, any gguf should work named to see above.

### Note on OCR
Windows OCR requires the language to be installed on the system.
